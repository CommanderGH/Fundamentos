TALLER DE JAVA SCRIPT

A continuación está la solución de los cuatro puntos del Taller de JavaScript. Cada uno de los recuadros es la solución de un punto del 1 al 4 empezando desde arriba y de forma consecutiva. El ejercicio de compensación fue añadido en el Punto 3 (tercer recuadro) , al cual se le agregó un campo de contraseña (el cual no está incluido en el enunciado original de punto 3), al cual se le hacen todas las verificaciones requeridas en el ejercicio de compensación. Cada punto tiene implementado el sistema de notificaciones tostada de SweetAlert2 el cual le da más estilo a los mensajes de alerta que se implementaron en los diferentes ejercicios. Cada ejercicio tiene funcionalidades adicionales a las pedidas en el taller, así que analice cada punto con atención. El código fue hecho con HTML, CSS y JavaScript puro, lo único que importé fue la librería de SweetAlert2, de resto todo es hecho desde cero.

      Formulario de Verificación

      Ingrese su nombre, su apellido y su correo en el siguiente formulario. Ambas entradas tienen diferentes requerimientos para ser válidas. Utilice el botón "Validar Entradas" para verificar si son correctas o para obtener información de los criterios que aún no han sido cumplidos.



      Los criterios que deben cumplir las entradas son los siguientes:

      • Deben haber almenos dos palabras en el campo "Nombre y Apellido" (correspondientes al nombre y al apellido).
      • El correo debe contener el caracter "@".
      • El correo debe contener el caracter ".".
      • El caracter "." debe estar después del caracter "@".
      • La contraseña debe tener entre 10 y 20 caracteres, contener al menos un número, una letra en mayúscula, una letra en minúscula, y un caracter especial.
